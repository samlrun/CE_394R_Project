0.1 README for Correlation Between Crash Severity and Crash Type

This article was written by Samuel Runyan in 2021. Samuel is a Civil Engineering student at Brigham Young University. This article fulfills a requirement for the CE 594R class on data science for engineers.

The purpose of this article is to answer the question, “Is there a significant correlation between the severity of vehicle crashes and the manner of collision?” The research and data presented in this article aim to answer this question.
